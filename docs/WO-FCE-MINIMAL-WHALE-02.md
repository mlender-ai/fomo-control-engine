## 킬존 대조와 위생 필요성

`docs/FCE-MOAT-STRATEGY-01.md` §2의 범용 차트 기능 경쟁을 확장하지 않는다. 이미 수집·표시하는
확정 OHLCV와 Hyperliquid 체결을 미니멀 화면에서 올바르게 읽게 만드는 **데이터 오표시·가독성
장애 수리**다. 신규 감지기·신규 지표·매매 판단 입력은 추가하지 않는다.

## 진단 (코드 확정)

- `LivePositionCockpit`은 미니멀 모드에서도 `chartTimeframe=4h`를 보유하지만 선택 UI는 프로의
  `PatternTimeframeMatrix`에만 있어 미니멀에서는 4시간봉으로 고정된다.
- `PriceLevelOverlay.priceLinesForAnalysis`가 현재가를 2px 불투명 실선으로 만들고,
  `PositionCandlestickChart`가 미니멀에도 그대로 렌더해 사용자가 고래 LIVE 선으로 오인한다.
- 고래 SVG는 `title` hover만 제공해 터치·클릭으로 지갑/금액을 확인할 수 없다.
- `kind=entry`는 채움, `kind=exit`는 빈 삼각형으로 표현하지만 차트 가까이에 범례가 없어 검증
  상태의 차이처럼 보인다. 후보 지갑 투명도까지 겹쳐 의미가 더 불명확하다.
- 차트 분석은 도메인 불변에 따라 확정 캔들만 사용한다. 현재가 한 점으로 미완성 OHLC를
  조작해 가짜 라이브 캔들을 만들 수 없으므로, 더 짧은 확정 시간봉과 현재가 별도 표기가
  정직한 해결이다.

## 작업

### 1. 미니멀 확정 시간봉 전환

- 미니멀 차트에 `15분 / 1시간 / 4시간 / 12시간 / 일봉` 전환 버튼을 노출한다.
- 기존 `chartTimeframe` 상태와 `/chart-analysis` 호출을 그대로 재사용한다.
- 로딩 중에도 직전 차트를 보존하고 선택 시간봉을 명시한다.

### 2. 현재가·캔들 의미 분리

- 미니멀 차트의 불투명 현재가 가로 실선을 제거한다.
- 헤더에 `현재가` 숫자와 `확정 캔들만 표시` 문구를 노출한다.
- 미완성 봉 OHLC를 추정하거나 현재가로 가짜 캔들을 만들지 않는다.

### 3. 고래 체결 마커와 상세

- 모든 롱/숏 체결 삼각형을 같은 채움 규칙으로 통일한다. 방향은 ▲/▼와 녹색/적색으로,
  행동은 `+ 진입·증액 / − 감액·청산`으로 분리한다.
- 28일 검증 통과 지갑이 포함된 그룹만 외곽 링으로 강조하고 범례에 의미를 적는다.
- 마커를 클릭/키보드 선택하면 체결 시각, 지갑 별칭·축약 주소, 방향·행동, 금액, 체결가,
  검증 상태를 실제 `items` 데이터로 표시한다.
- 우측 LIVE 마커도 동일 규칙을 사용하며 실제 체결 시각을 보존한다.
