# Telegram Position Monitoring

실계좌 주문을 실행하지 않는 read-only 포지션 관제 명령이다.

## Commands

- `/positions`, `/p`: 열린 포지션을 한 메시지로 요약한다.
- `/positions_full`, `/pf`: 현재 보유 중인 모든 포지션을 심볼별 상세 메시지로 보낸다.
- `/p SYMBOL`: 한 포지션만 상세 조회한다.
- `/flow SYMBOL`: 한 심볼의 파생 수급 원자료를 조회한다.

`/positions` 결과의 `전체 상세` 버튼은 `/positions_full`과 같은 조회를 실행한다.

## Detailed position message

각 메시지는 현재 방향·레버리지·PnL·건강도와 함께 다음 정보를 제공한다.

- 현재 액션 플랜과 무효화·익절 감시 가격
- 모듈별 1줄 판정과 종합 브리핑
- OI·펀딩·쏠림
- 현물·선물 CVD 기반 자금 흐름 상태와 자연어 해석
- 분석 출처와 기준 시각

자금 흐름 설명은 관측의 의미를 밝히며 매수·매도 지시나 방향 확정으로 사용하지 않는다. 한 심볼의 상세 계산이 제한시간을 넘겨도 다른 보유 포지션 조회는 계속 진행한다.
